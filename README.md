# ComputerVision

In this project of computer vision, we have implemented the image captioning model with the help of two architectures, First architecture includes BLIP, YOLO along with Sentence Transformer which gives BLEU-1 score of 0.517 while the second architecture includes BLIP, CLIP, YOLO along with FAISS search which gives BLEU-1 score of 0.774. This is the comparative study of the above architecture models that we have stacked to build upon the recent high quality models.
